# Guides

Pentaho Big Data Guides

# CREATE TABLE
```
CREATE TABLE test_table ( 
key    String, 
call_date    String, 
source_number    String, 
call_month    String, 
call_year    String, 
day_of_week    String, 
area_code    String,
state    String, 
country    String, 
time_zone    String, 
weekday    String, 
num_calls    String);
```

# DROP TABLE
```
drop table test_table;
```

### Select 
```
select * from test_table;
```
